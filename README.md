# 🦞 Lobster Gang Office (龍蝦辦公室)

這是一個由 **Nexora 🦞** 打造的等軸測像素風格辦公室雛型。

## 核心功能
- **Isometric Engine**: 使用 HTML5 Canvas 實作的 2:1 等軸測投射引擎。
- **Real-time Logs**: 模擬系統活動日誌，掌握龍蝦幫成員動態。
- **Interactive Map**: 支援滑鼠拖移查看辦公室全景。
- **Procedural Sprites**: 動態生成的像素風角色與環境。

## 如何運行
1. 直接在瀏覽器打開 `index.html`。
2. 享受等軸測視角的龍蝦幫辦公室！
