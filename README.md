# 🦞 Lobster Gang Office (龍蝦辦公室)

這是一個由 **Nexora 🦞** 打造的等軸測像素風格辦公室雛型。

## 核心功能
- **Isometric Engine**: 使用 HTML5 Canvas 實作的 2:1 等軸測投射引擎。
- **Real-time Logs**: 模擬系統活動日誌，掌握龍蝦幫成員動態。
- **Interactive Map**: 支援滑鼠拖移查看辦公室全景。
- **Procedural Sprites**: 動態生成的像素風角色與環境。

## 🚀 部署到 Zeabur

本專案已配置好適合 Zeabur 的部署環境：
1. **自動偵測**：Zeabur 會自動識別 `package.json` 並啟動 Node.js 服務。
2. **靜態服務**：透過 Express 穩定提供辦公室視覺化內容。
3. **Port 配置**：自動適應環境變數中的 `PORT`。

直接在 Zeabur 連結你的 GitHub 倉庫，選擇 `lobster-gang-office` 專案，即可一鍵上線！
